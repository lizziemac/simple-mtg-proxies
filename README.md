# Simple MtG Proxies

Simple site that converts your MtG cards to essentially plaintext cards with no art to make
printer-friendly, low ink proxies.

# On Initial Download

## Install depedencies

```sh
npm install
```


# Run Locally

```sh
npm run start
```

## Running Tests

```sh
npm test
```

## Running Linting

```sh
npm run lint
```

## Compile for Production

```sh
npm run build
```
